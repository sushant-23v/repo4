# Desxto Coffee

A warm & cozy coffee shop landing site built with React, React Router, Tailwind CSS and Vite.

## Getting Started

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- React Router DOM 6
- Tailwind CSS 3
- Vite 5
- Axios
