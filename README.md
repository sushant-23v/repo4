# Desxto Coffee

A warm & cozy coffee shop landing site built with React, React Router, Tailwind CSS, and Axios.

## Getting Started

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- React Router v6
- Tailwind CSS
- Axios
- Vite

## Brand

**Desxto Coffee** — Where every cup feels like home. Warm espresso browns, soft cream, and amber highlights.
