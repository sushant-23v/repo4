# Goa Escape — Romantic 8-Day Trip Planner

A React + Tailwind website presenting a complete romantic Goa getaway plan for Sushant & Sonali, traveling from Pune (July 30 – August 6, 2026).

## Trip Snapshot
- Travelers: Sushant (36) and Sonali (33), both non-vegetarian
- Dates: 30 Jul 2026 – 06 Aug 2026 (7 nights, 8 days)
- Origin: Pune
- Vibe: romantic, beach relaxation, nightlife, food experiences
- Budget: INR 1,00,000 total for both travelers

## Getting Started
```bash
npm install
cp .env.example .env
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Tech Stack
- React 18 + Vite
- React Router v6 (createBrowserRouter)
- Tailwind CSS
- Axios
