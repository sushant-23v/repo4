# Desxto Coffee

A warm and cozy coffee shop landing website built with React, Vite, React Router, and Tailwind CSS.

## Getting Started

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite 5
- React Router DOM 6
- Tailwind CSS 3
- Axios

## Brand

**Desxto Coffee** — Warm browns, creamy backgrounds, and soft amber accents for a cozy neighborhood coffee shop vibe.
